# restapi-example
A reference implementation of GigaSpaces Unified Grid Manager/REST API client usage.

To use:

1. Start a XAP Manager locally, as documented here: https://docs.gigaspaces.com/xap/12.1/admin/xap-manager-rest.html
2. Use REST API to start a GSC (make an HTTP call).
3. Execute the JAX-RS code stored in this repo (make an HTTP call).
